Unicode 引用

Machine's Mi-iconfont 使用步骤如下：

第一步：拷贝项目下面生成的@font-face


 @font-face {
   font-family: Machine-Mi-iconfont;
   src: url(machine-iconfont.woff) format("woff");
}

第二步：定义使用 iconfont 的样式

.icon {
  display: inline-block;
  font-family: Machine-Mi-iconfont !important;
  font-style: normal;
  font-weight: normal;
  line-height: 1;
 }

第三步：挑选相应图标并获取字体编码，应用于页面


<span class="iconfont">&#x628;</span>
